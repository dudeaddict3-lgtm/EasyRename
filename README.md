# 🚀 大厂批量改名神器 (BatchRenamer Pro)

一个基于 Python 开发的专业级文件重命名工具，支持 GUI 界面与撤销功能。

## ✨ 核心功能
* **多种规则**：支持添加前缀、文字替换、序列化重命名（001, 002...）。
* **安全预览**：在执行前实时预览改名效果，防止误操作。
* **后悔药 (Undo)**：一键撤销刚才的改名操作，容错率极高。
* **独立运行**：已打包为 EXE，无需 Python 环境即可运行。

## 🛠️ 技术架构
本项目采用 **MVC 架构**，逻辑与界面完全解耦：
* `src/rules.py`: 定义改名算法（策略模式）。
* `src/renamer.py`: 核心处理引擎。
* `gui_main.py`: 基于 tkinter 的图形界面。
